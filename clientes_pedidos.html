Clientes y sus Pedidos

Consulta SQL – Sistema de Análisis de Ventas

Código SQL

SELECT
    c.ClienteID,
    c.NombreCliente,
    p.PedidoID,
    p.FechaPedido,
    p.Estado
FROM Clientes c
LEFT JOIN Pedidos p
    ON c.ClienteID = p.ClienteID
ORDER BY c.NombreCliente, p.FechaPedido;
