Productos sin Ventas

Consulta SQL – Sistema de Análisis de Ventas

Código SQL

SELECT
    pr.ProductoID,
    pr.NombreProducto,
    pr.Categoria,
    pr.Stock
FROM Productos pr
LEFT JOIN DetallesPedido dp
    ON pr.ProductoID = dp.ProductoID
WHERE dp.ProductoID IS NULL;
